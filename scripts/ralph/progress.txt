## Codebase Patterns
- Root `.gitignore` has `*.json` — must add negation patterns (`!package.json`, `!tsconfig.json`, `!tsconfig.*.json`) to commit JSON files
- Node package at `./node` relative to repo root, uses ESM (`"type": "module"`)
- TypeScript strict mode with NodeNext module resolution
- All TS imports use `.js` extension (NodeNext requires it)
- API field names preserve wire format casing, function params are camelCase
- Token cache uses Date.now() in milliseconds (Python uses time.time() in seconds)
- Operation endpoints add `fetch_current_account: "no_value"` header when fetchCurrentAccount is true
- Files outside `src/` (like test.ts) need a separate tsconfig with `rootDir: "."`
